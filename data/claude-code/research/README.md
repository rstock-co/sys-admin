# Claude Code Release Research

Deep-dive research files for specific Claude Code releases.

## File Naming

Files are named by version with dashes: `2-0-54.md`, `2-0-72.md`, etc.

## When These Are Created

Research files are created when you ask for deep research on a specific release:
- "Research 2.0.54 in depth"
- "Deep dive on release 2.0.72"
- "Tell me more about the Chrome integration"

## What They Contain

Each file includes:
- Official changelog details
- External research (blogs, docs, community)
- Practical usage examples
- Relevance to YOUR specific setup (based on relevance.md)
- Source links

## Relationship to Other Files

| File | Purpose |
|------|---------|
| `changelog.md` | Comprehensive log of ALL releases |
| `relevance.md` | Filtered view of what matters to YOU |
| `research/*.md` | Deep dives on specific releases |
